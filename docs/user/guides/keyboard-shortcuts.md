# Keyboard Shortcuts

This guide provides a complete list of keyboard shortcuts for the Obsidian CLI Ops TUI.

## Global Shortcuts

These shortcuts work on almost every screen.

| Key(s)         | Action                   |
|----------------|--------------------------|
| `q` / `Ctrl+Q` | Quit the application     |
| `Esc`          | Go back / Close a dialog |
| `?`            | Open this Help screen    |

## Main Menu (Home Screen)

| Key | Action                          |
|-----|---------------------------------|
| `v` | Open **V**ault Browser          |
| `n` | Open **N**ote Explorer          |
| `g` | Open **G**raph Visualizer       |
| `s` | Open **S**tatistics Dashboard   |

## Vault Browser

| Key(s)       | Action                                |
|--------------|---------------------------------------|
| `↑` / `k`    | Move selection up                     |
| `↓` / `j`    | Move selection down                   |
| `Enter`      | Open selected vault (Note Explorer)   |
| `d`          | **D**iscover new vaults in iCloud   |
| `g`          | View **G**raph for selected vault     |
| `s`          | View **S**tats for selected vault     |
| `r`          | **R**efresh the vault list            |

## Note Explorer

| Key(s)           | Action                                   |
|------------------|------------------------------------------|
| `↑` / `k`        | Move selection up                        |
| `↓` / `j`        | Move selection down                      |
| `Enter`          | View full note (Future Feature)          |
| `/` / `Ctrl+F`   | Focus **S**earch box                     |
| `s`              | Cycle through **S**ort options           |
| `r`              | **R**efresh the note list                |

## Graph Visualizer

| Key(s)    | Action                               |
|-----------|--------------------------------------|
| `↑` / `k` | Move selection up                    |
| `↓` / `j` | Move selection down                  |
| `Enter`   | View **N**eighborhood of selected note |
| `h`       | Toggle **H**ubs view                 |
| `o`       | Toggle **O**rphans view              |
| `c`       | Toggle **C**lusters view             |
| `n`       | View full details for selected **n**ote|
| `r`       | **R**efresh the graph data           |

---
**Note on Saving:** There is no manual "save" shortcut because all data is either read directly from the filesystem or saved to the internal database automatically.
