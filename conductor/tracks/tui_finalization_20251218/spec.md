# Track Spec: TUI Finalization

## Goal
Finalize the v2.1.0 release by completing the Note Explorer and Graph Visualizer screens in the Textual-based TUI.

## Scope
- **Note Explorer**: Interactive screen for searching, filtering, and previewing notes.
- **Graph Visualizer**: Interactive screen for visualizing the knowledge graph using NetworkX and Rich.

## Requirements
- Note Explorer should support real-time search and metadata display.
- Graph Visualizer should highlight hubs and orphans.
- Navigation between screens should be seamless using Vim-style keys and shortcuts.
