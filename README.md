# Obsidian CLI Ops

A command-line interface (CLI) tool for managing a federated Obsidian vault system.

## Features
*   **Sync:** Keep themes/hotkeys consistent across vaults.
*   **Install:** Add plugins from GitHub to one or all vaults.
*   **Audit:** Check vault structure.

## Installation
1.  Symlink \`src/obs.zsh\` to \`~/.config/zsh/functions/obs.zsh\`.
2.  Autoload \`obs\` in your \`.zshrc\`.
3.  Configure \`~/.config/obs/config\`.
